# 🧮 Calculadora em Python

Projeto de uma calculadora desenvolvida em Python utilizando Programação Orientada a Objetos (POO), com operações matemáticas básicas e avançadas.

O objetivo do projeto foi praticar:

* modularização
* funções
* tratamento de erros
* Programação Orientada a Objetos
* organização de código
* interação via terminal

---

## 🚀 Funcionalidades

### Operações com dois números

* Soma
* Subtração
* Multiplicação
* Divisão
* Potência
* Resto da divisão
* Divisão inteira
* Porcentagem

### Operações com um número

* Raiz quadrada
* Valor absoluto
* Logaritmo
* Fatorial
* Seno
* Cosseno
* Tangente

### Extras

* Histórico de operações
* Tratamento de erros
* Interface no terminal
* Classe `Calculadora`
* Métodos organizados por responsabilidade

---

## 🛠️ Tecnologias utilizadas

* Python 3
* Biblioteca `math`

## ▶️ Como executar

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Entre na pasta do projeto:

```bash
cd Projeto_Calculadora
```

Execute o arquivo principal:

```bash
python main.py
```
