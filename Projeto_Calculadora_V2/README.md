# 🧮 Calculadora em Python

Projeto de uma calculadora desenvolvida em Python com operações matemáticas básicas e avançadas.

O objetivo do projeto foi praticar:
- modularização
- funções
- tratamento de erros
- organização de código
- interação via terminal

---

## 🚀 Funcionalidades

### Operações com dois números
- Soma
- Subtração
- Multiplicação
- Divisão
- Potência
- Resto da divisão
- Divisão inteira
- Porcentagem

### Operações com um número
- Raiz quadrada
- Valor absoluto
- Logaritmo
- Fatorial
- Seno
- Cosseno
- Tangente

### Extras
- Histórico de operações
- Tratamento de erros
- Interface no terminal
- Código dividido em módulos

---

## 🛠️ Tecnologias utilizadas

- Python 3
- Biblioteca `math`
